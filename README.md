## Overview

My personal portfolio as a **Cybersecurity & IT Specialist**: experience, skills, certifications and real-world case studies. Built as a fast, static, single-page site — no framework, no tracking, self-hosted fonts.

## Features

- **Bilingual IT / EN** with an instant language switch (choice remembered locally)
- **Cinematic dark design** with smooth reveal animations and a responsive layout
- **Self-hosted fonts** (Fraunces, Inter, Space Mono) — zero third-party requests
- SEO-ready: meta tags, Open Graph, `schema.org` structured data, `sitemap.xml`
- Downloadable CV (IT / EN)

## Tech stack

Plain **HTML5**, **CSS3** and **vanilla JavaScript**.​ No build step, no dependencies.

## Run locally

```bash
# just open the file — or serve it
python -m http.server 8000
# then visit http://localhost:8000
```

## Structure

```
index.html          # all sections
assets/css/         # style.css (theme tokens in :root)
assets/js/          # main.js (i18n, animations, nav)
assets/fonts/       # self-hosted woff2
assets/img/ · doc/  # images, badges, CV
```
